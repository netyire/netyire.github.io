# 🌸 Welcome to Netyire's Web Playground!

Hi there! This space hosts a collection of single-file, interactive static HTML projects. Since they are built to run directly in your web browser, there is no installation or server setup required—just click and explore!

---

## 🚀 Projects Hosted Here

### 📅 [Daily Life Tracker (DLT) v3.64.1](https://netyire.github.io/index.html)
A comprehensive, privacy-first personal assistant and productivity suite.
* **What it does**: Helps you keep track of your daily notes, activities, habits, and tasks in one place.
* **Why it's cool**: It runs completely locally in your browser! All your data is securely saved on your own device using browser storage (IndexedDB), with optional cloud backup and restore syncing via Google Drive.
* **Features**: Dynamic widgets, voice transcription logs (with advanced filters and noise suppression settings), and a sleek dark-themed responsive interface.

### 🌎 [Chapter 3 - Earth: Grounding Your Thinking](https://netyire.github.io/c3.html)
An interactive study guide and learning companion.
* **What it does**: A beautiful, highly structured companion for Chapter 3 of the acclaimed book ***"The 5 Elements of Effective Thinking"*** by Edward B. Burger and Michael Starbird.
* **Why it's cool**: It turns book concepts into structured, collapsible cards, key takeaways, and "illustrative bridges" comparing historical figures and literature (like Robert Frost's poetry) to make deep critical thinking easy to grasp.
* **Features**: Multiple responsive visual themes (including Ubuntu, E-Ink, and Pearl), collapsible sections for focused reading, and simple, friendly explanations.

## 🛠️ How to Run Locally

Since these are static pages, you can run them directly on your machine:
1. Clone this repository:
   ```bash
   git clone https://github.com/netyire/netyire.github.io.git
   ```
2. Open either [DLT3.64.1.html](index.html) or [c3.html](c3.html) in your favorite web browser (Chrome, Firefox, Safari, Edge).
